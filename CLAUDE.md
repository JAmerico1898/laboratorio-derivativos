# Derivativos Lab

Educational web app for teaching financial derivatives to MBA students at COPPEAD/UFRJ.

## Conventions
- All UI content in Portuguese (financial domain-specific)
- UI strings centralized in `src/lib/strings.ts`
- Dark theme (#0a0f1a background)
- Module routes: /termos, /futuros, /swaps, /opcoes, /credito, /embutidos
